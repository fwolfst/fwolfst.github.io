list all crontabs

Nov 1, 2020 [ ]

Often enough, we search a cron entry and are not sure which user is issuing it, or we doubt if there is a crontab entry present (or missing).

In this case, outputting all manually registered crontabs can be handy:

for user in $(cut -f1 -d: /etc/passwd); do crontab -u $user -l; done

(you need extended permissions for that command).
